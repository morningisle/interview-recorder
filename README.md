# 面试记录与复盘工具 (Interview Recorder & Review)

这是一款专为求职者设计的纯前端面试管理工具，帮助您记录面试经历、通过 AI 辅助解析文字稿、建立个人题库并进行深度复盘分析。

## 🌟 核心特性

- **完全私密**：所有数据均存储在您的本地浏览器（localStorage），无需注册，不上传服务器，隐私安全受法律级别保护。
- **智能题库**：支持从面试录音文字稿中自动识别 Q&A，并自动关联公司、职位。
- **自定义类别**：完全自主的问题分类体系，支持手动添加、编辑和批量修改。
- **错题本模式**：一键查看所有已记录“优化后回答”的题目，聚焦复盘精华。
- **可视化分析**：通过 Canvas 绘制面试趋势图，直观展现求职进度。
- **数据流转**：支持导出 Markdown 格式的精美复盘报告，以及 JSON 格式的数据备份包。

## 🚀 如何使用

1. **本地运行**：
   - 直接双击 `index.html`（部分浏览器可能因安全限制无法加载模块）。
   - **推荐**：在文件夹内运行 `python -m http.server 8000`，然后在浏览器访问 `http://localhost:8000`。
   - 或者使用 VS Code 的 `Live Server` 插件。

2. **部署分享**：
   - 本项目为纯静态网页，可直接上传至 GitHub Pages、Vercel 或 Netlify 免费托管。

## 📂 数据导入与导出

- **导出报告 (Ctrl + E)**：生成 Markdown 文件，适合在手机或 Word/WPS 中阅读复盘。
- **数据备份 (Ctrl + B)**：生成 JSON 文件，包含所有数据，可用于在不同设备间迁移或分享给好友。
- **导入数据**：选择他人分享的 `.json` 备份包，即可一键合并面试记录和题库。

## 🛠️ 技术栈

- 纯原生 HTML5 / CSS3 (BEM 命名规范)
- 现代 JavaScript (ES6+ 模块化)
- Canvas API (自定义图表绘制)
- 响应式设计 (支持 320px - 1920px 屏幕)
